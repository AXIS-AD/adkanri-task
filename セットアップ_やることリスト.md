# セットアップ やることリスト

## 事前準備（5分）

- [ ] **Chatwork APIトークン**を取得
  - Chatwork → 設定 → API → 「APIトークンを発行」
- [ ] **通知先ルームID**を確認
  - ルームURLの `rid` の後の数字（例: rid123456789 → 123456789）
- [ ] **担当者のユーザーID**を確認
  - ルームメンバーをクリック → URLの `account_id=○○○`
- [ ] `設定値メモ.md` に値を記入

---

## STEP 1: GASプロジェクト作成（2分）

- [ ] [script.google.com](https://script.google.com/) を開く
- [ ] 「新しいプロジェクト」をクリック
- [ ] プロジェクト名を「TOアド管理依頼」に変更

---

## STEP 2: コード貼り付け（3分）

- [ ] 左の **Code.gs** を開く → 中身を全削除
- [ ] このフォルダの **Code.gs** を開く → 全選択（Ctrl+A）→ コピー
- [ ] GASの Code.gs に貼り付け
- [ ] 左の「+」→「HTML」→ ファイル名を **index** に
- [ ] このフォルダの **index.html** を開く → 全選択 → コピー
- [ ] GASの index に貼り付け

---

## STEP 3: 設定値の置き換え（5分）

- [ ] GASの Code.gs を開く
- [ ] 7行目: `YOUR_CHATWORK_API_TOKEN` → 実際のAPIトークン
- [ ] 8行目: `YOUR_CHATWORK_ROOM_ID` → 実際のルームID
- [ ] 11行目: `ALLOWED_EMAIL_DOMAINS` → 必要に応じて変更（そのままでOK）
- [ ] 14〜22行目: `USER_ID_1` / `USER_ID_2` → 担当者のユーザーID
- [ ] 25行目: `YOUR_USER_IDS_COMMA_SEPARATED` → 営業時間外の通知先（例: `'123,456,789'`）

---

## STEP 4: デプロイ（2分）

- [ ] 右上「デプロイ」→「新しいデプロイ」
- [ ] 種類: 「ウェブアプリ」
- [ ] **次のユーザーとして実行: ユーザー**（必須）
- [ ] アクセス: 組織内 or リンクを知っている全員
- [ ] 「デプロイ」→ 初回は「アクセスを承認」
- [ ] **URLをコピー**して保存

---

## STEP 5: 動作確認（2分）

- [ ] AXISアドレスでログインした状態でURLを開く
- [ ] 「依頼者: ○○○@axis-ads.co.jp」と表示される
- [ ] テスト依頼を1件送信
- [ ] チャットワークにタスクが作成される
- [ ] タスク一覧タブで進捗更新できる

---

## 完了

依頼ツールのURLを共有して運用開始！
