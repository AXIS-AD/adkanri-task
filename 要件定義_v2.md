# やりたいこと

1. **依頼者名**：依頼者のGoogleアドレスから氏名を自動取得
2. **タスク依頼**：わかりやすいUIで依頼フォームを実施
3. **タスク化**：チャットワークにタスクとして登録
4. **進捗管理**：依頼ツール上でタスクの進捗を記載・更新できる

# アーキテクチャ

- **スプシ廃止**：依頼データ・氏名マスタは不要
- **ストレージ**：PropertiesService（task_id ↔ 進捗の紐づけ）
- **タスク一覧**：Chatwork API から取得
- **認証**：GAS「ユーザーとして実行」で Session.getActiveUser().getEmail()

- 未対応
- 対応中
- 完了（Chatworkのタスクも完了に同期）
