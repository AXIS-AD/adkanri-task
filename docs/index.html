アド管理タスク 使い方ガイド

広告事業部のタスク依頼・管理をスムーズに行うためのツールです

目次

ツール概要 新規依頼の出し方 依頼状況の確認 タスク管理（管理者） 定期・長期タスク 出退勤タスク報告 管理設定 よくある質問

ツール概要

このツールは、Chatworkと連携したタスク依頼・管理ツールです。依頼者はフォームからタスクを依頼し、管理者はダッシュボードでタスクの進捗を一元管理できます。

👤
一般ユーザー
  • 新規依頼の送信
  • 自分が出した依頼の進捗確認
👑
管理者（Admin）
  • 全タスクの一覧・ステータス管理
  • 定期タスク・長期タスクの追加
  • 出退勤タスク報告の送信
  • ユーザーの権限管理

✏️ 新規依頼の出し方

1
依頼内容を選択

アカウント関連、リンク発行、スプシ関連など、該当するカテゴリカードをクリックします。

2
詳細カテゴリを選択

選んだカテゴリに応じた小分類が表示されます。該当するものを選んでください。

3
詳細を入力

案件名やアカウント名など、必要な情報をフォームに入力します。項目はカテゴリによって変わります。

4
送信

「送信する」をクリックすると、Chatworkにタスクが自動作成されます。

💡 ヒント：営業時間外（18:00〜翌9:00）に送信した場合、タスクは自動的に翌営業日に振り分けられます。

📅 依頼状況の確認

自分が送信した依頼の現在のステータスを確認できます。

項目説明
内容タスクのタイトル（管理者がタイトルを編集した場合はそちらが反映）
担当者Chatwork上の実際の担当者名
期日タスクの期日（クリックで変更可能）
ステータス未着手 / 着手中 / 待ち / 完了
💡 ヒント：タスクの行をクリックすると、詳細内容がポップアップで確認できます。完了済みのタスクは自動的に非表示になります。

🌟 タスク管理（管理者のみ）

ダッシュボード

タスク管理を開くと、まず全体のサマリーが表示されます。

  • 全体統計：全タスク数、未着手、着手中、待ちの件数
  • 期限バッジ：今日が期日 / 期限切れの件数
  • メンバーカード：各メンバーの担当タスク数（クリックで個人タスク一覧へ）

タスク一覧

担当者やカテゴリを左サイドバーで絞り込み、カード or リストで表示できます。

操作方法
ステータス変更カード上のステータスバッジをクリック → ドロップダウンから選択
優先度変更ポップアップ内の優先度セレクトから変更
メモ追加カード上のメモ欄に直接入力（自動保存）、またはポップアップ内の保存ボタン
期日変更期日をクリック → カレンダーから日付選択
タイトル編集ポップアップの鉛筆アイコンをクリック → 編集 → Enter
完了にするポップアップ内の「✅ 完了にする」ボタン → コメント入力（任意）→ 確定
💡 ヒント：タスクを完了にすると、Chatwork上のタスクも完了になり、依頼者に完了通知が送信されます。

🔁 定期タスク・長期タスク

カテゴリサイドバーから「定期タスク」「長期タスク」を選ぶと、手動でタスクを追加できます。

定期タスク

+
追加方法

「+ 定期タスクを追加」ボタンをクリックし、タスク名・頻度・期日・時間を入力して保存します。

頻度タイプ説明
間隔毎日 / 毎週 / 隔週 / 毎月 から選択
曜日特定の曜日を複数選択可能（例：月・水・金）
日付毎月の特定日を複数選択可能（例：1日・15日）
💡 ヒント：定期タスクのカードをクリックすると、TODO管理や詳細編集が行えます。時間を設定すると、出勤報告にも表示されます。

長期タスク

期限が長いプロジェクト型のタスクを管理します。追加方法は定期タスクと同様です。

📋 出退勤タスク報告

タスク管理画面のヘッダーにある「📋 報告」ボタンから、Chatworkにタスク報告を送信できます。

出勤報告（☀️）

  • 今日の期日 + 期限切れのタスクが自動でリストアップされます
  • 定期タスクは当日の頻度に該当するもののみ表示
  • カテゴリ（定期タスク / 数字合わせ / つなぎこみ / その他 / 待ち）に自動分類

退勤報告（🌙）

  • 出勤報告に含めたタスクの現在のステータスを表示
  • ステータスアイコン：✅ 完了 / ❌ 未着手 / 🔼 着手中 / ⏳ 待ち
  • 出勤報告になかったが完了したタスクは「差し込みタスク」として表示
💡 ヒント：送信前にテキストを編集できます。内容を確認してから「送信」ボタンを押してください。

⚒️ 管理設定（管理者のみ）

ユーザーの権限を管理します。

権限できること
Adminタスク管理・管理設定の全機能にアクセス可能
User新規依頼の送信と依頼状況の確認のみ
💡 ヒント：初回ログインしたユーザーは自動的にUserとして登録されます。管理者が「Admin」に変更できます。

よくある質問

ログインできません

Google アカウントで @axis-hd.co.jp ドメインのみ使用可能です。個人Googleアカウントではログインできません。ブラウザのポップアップブロックを無効にしてお試しください。

依頼を送信したのにタスクが見つかりません

依頼はChatwork上でタスクとして作成されます。「依頼状況」タブで確認してください。反映に少し時間がかかる場合があります。

タスクのステータスが変わりません

ステータス変更後、左上の「更新」ボタンを押すと最新の状態が反映されます。

担当者が違う人になっています

Chatwork上の実際のタスク担当者が表示されます。Chatworkでタスクを完了した人のタスクは自動的にスキップされ、次の担当者が表示されます。

期限切れのアラートはいつ届きますか？

毎営業日の朝10時（JST）に、期日が3日以上過ぎたタスクのアラートがChatworkに通知されます。土日はアラートが送信されません。

テーマを切り替えたい

画面右上の 🌙 アイコンをクリックすると、ダークモード ↔ ライトモードを切り替えられます。

不明点やバグ報告は、Chatwork「広告事業部チャット」までお問い合わせください。
