# スクリプト プロパティの設定

GAS の「プロジェクトの設定」→「スクリプト プロパティ」で以下を追加します。

## 追加するプロパティ

| プロパティ | 値の例 |
|-----------|--------|
| `CHATWORK_API_TOKEN` | あなたのAPIトークン |
| `CHATWORK_ROOM_ID` | 123456789 |
| `ALL_USER_IDS` | 123,456,789 |
| `ASSIGN_MAP_JSON` | （下記参照） |

---

## ASSIGN_MAP_JSON の値

カテゴリごとの担当者（ChatworkユーザーID）を JSON 形式で指定します。

```
{"アカウント関連（作成・紐づけ・エラー）":"123","リンク発行（ナハト・AXIS・DIO）":"123","スプシ関連（精査用シート・分析シート・運用シート）":"456","新規案件":"456","その他/エラー関連":"456","CSV保管修正":"456","単価変更":"456","キャップ通知依頼":"456","新規オファー追加（アカ開設・単価登録・リンク発行などまとめて依頼）":"456"}
```

※ `123` / `456` を実際の担当者のユーザーIDに置き換えてください。
※ 1行で、改行なしで入力します。
