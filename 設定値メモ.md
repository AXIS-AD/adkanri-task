# 設定値メモ

Code.gs を編集する前に、ここに値をメモしておくと便利です。

## Chatwork の取得方法

| 項目 | 取得方法 |
|------|----------|
| APIトークン | Chatwork → 右上アイコン → 設定 → API → 「APIトークンを発行」 |
| ルームID | 通知先ルームのURLの数字部分<br>例: `https://www.chatwork.com/#!rid123456789` → `123456789` |
| ユーザーID | ルーム内でメンバーをクリック → URLの `account_id=○○○` の数字 |

---

## 記入欄（Code.gs に貼り付ける値）

```
CHATWORK_API_TOKEN = '                    '  ← APIトークン
CHATWORK_ROOM_ID   = '                    '  ← ルームID

ALLOWED_EMAIL_DOMAINS = ['axis-ads.co.jp', 'axis-hd.co.jp']  ← 必要に応じて変更

USER_ID_1 =            ← アカウント・リンク担当者のChatworkユーザーID
USER_ID_2 =            ← その他担当者のChatworkユーザーID

ALL_USER_IDS = '       '  ← 営業時間外の通知先（カンマ区切り 例: 123,456,789）
```
